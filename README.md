# Luis Porto - Portfolio

Portfolio profesional desarrollado para mostrar proyectos, experiencia y servicios relacionados con el desarrollo de software y soluciones empresariales.

## 🚀 Acerca del Proyecto

Este sitio web presenta mi experiencia como desarrollador Full Stack y fundador de Porto Soluciones, destacando proyectos, tecnologías y servicios enfocados en la creación de aplicaciones web modernas.

### Objetivos

* Presentar proyectos destacados.
* Mostrar experiencia técnica.
* Compartir servicios de desarrollo.
* Facilitar el contacto con clientes y colaboradores.

## 📸 Proyectos Destacados

### MiConsultorio

Sistema de gestión médica diseñado para optimizar la administración de consultorios y especialistas.

Características:

* Gestión de pacientes.
* Agenda médica.
* Historia clínica.
* Control de citas.
* Acceso desde múltiples dispositivos.

### Sistema de Riñas de Gallos

Plataforma especializada para la gestión de eventos, programación de enfrentamientos y seguimiento estadístico.

Características:

* Gestión de ejemplares.
* Programación de peleas.
* Estadísticas y reportes.
* Administración de eventos.

## ⚙️ Instalación

Clonar el repositorio:

git clone https://github.com/tu-usuario/luis-porto-portfolio.git

Ingresar al proyecto:

cd luis-porto-portfolio

Instalar dependencias:

npm install

Ejecutar en desarrollo:

npm run dev

Abrir:

http://localhost:3000
