# Imágenes del Portafolio

## Archivos requeridos:

### luis-porto.webp
- Fotografía profesional de Luis Porto
- Formato: WebP
- Dimensiones recomendadas: 800x800px (se mostrará con recorte circular)
- Se usa en la sección Hero con prioridad de carga (above the fold)

### og-image.png
- Imagen para compartir en redes sociales (Open Graph)
- Formato: PNG
- Dimensiones: 1200x630px
- Se usa en metadatos de SEO cuando se comparte el sitio en redes sociales

## Notas

- Mientras no existan estos archivos, se mostrará un placeholder visual
- El componente next/image maneja optimización automática (WebP/AVIF según navegador)
- La imagen del hero tiene prioridad de carga (no usa lazy loading) por estar en el viewport inicial
